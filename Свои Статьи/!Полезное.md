```python
#1 | Можно декорировать любую функцию, он будет запоминать результаты и при повторном вызове с теме же параметрами не будет ее вызывать 
from functools import lru_cache
@lru_cache
def fib(x):
    return fib(x-2)+fib(x-1) if x>1 else x

```
